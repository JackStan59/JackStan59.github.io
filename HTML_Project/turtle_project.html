Turtle Project

About The Project

This project uses the turtle module in python to create drawings and shapes on the screen using code commands to control the turtle cursor. The turtle can be moved around the screen, change colors, and draw various shapes based on the commands given in the code. This project is a fun way to learn programming concepts while creating visual art with code.
